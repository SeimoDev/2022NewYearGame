# 2022年新春解密
一个有趣的新年解密游戏.

## 在线游玩:

[点击这里](https://2022games.geekteam.top)


## 服务器要求
- Linux 服务器
- PHP:7.4+
- MySQL 5.6+/8.0.23+ or MariaDB 10.0.5+

## 安装

1. 新建数据库。
2. 将`` jiemi.sql `` 导入数据库。
3. 在`` config.php `` 填写数据库信息。
4. 在`` mail.php `` 填写邮件发信服务器信息。
